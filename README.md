# Shameem Haldoon P — Professional Portfolio

> **CEO & Entrepreneur** | TOMERCON DEVELOPERS | Kozhikode, Kerala, India

A modern, minimalist personal portfolio website built with HTML, CSS, and JavaScript — showcasing professional experience, leadership, and entrepreneurial ventures.

---

## 🌐 Live Demo

[View Portfolio](https://jafar1090.github.io/shammem-haldoon-porfolio-latest/)

---

## ✨ Features

- 🎨 **Premium Bento-style UI** — Modern glassmorphism design
- ✍️ **Typewriter Animation** — Dynamic role titles in the hero section
- 📦 **Professional Experience Cards** — Hover background images with smooth transitions
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- 🌟 **Scroll Reveal Animations** — Elements animate as you scroll
- 🖱️ **Mouse Glow Effect** — Interactive cursor glow
- 📬 **Contact Bar** — One-click email copy + LinkedIn + social links
