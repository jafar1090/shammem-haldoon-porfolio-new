# Shameem Haldoon P — Portfolio

Personal portfolio site for Shameem Haldoon P, CEO at TOMERCON DEVELOPERS, Kozhikode, Kerala.

Built with plain HTML, CSS, and JavaScript — no frameworks, no build tools.

---

## What's inside

- Hero section with a typewriter effect for job titles
- About section with a short background and stats
- Professional experience laid out as a bento-style card grid
- Contact bar with email, LinkedIn, and location links

---

## Tech

HTML, CSS, vanilla JavaScript. Google Fonts (Inter + Outfit) for typography, Font Awesome for icons.

---

## Run locally

Just open `index.html` in a browser. No setup needed.
